# webpack-dev-server
## 설치
```bash
npm init -y

npm install --save-dev babel-loader babel-core babel-preset-env

npm install --save-dev webpack webpack-dev-server

npm install --save-dev webpack-cli

```

### babel-loader 와 core 버전 맞춰주어야 정상 동작한다.

## 시작

### 개발
```bash
npm run devserver
```
### 배포빌드
```bash
npm run build
```
